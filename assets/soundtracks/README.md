Soundtrack assets go here.

Each game has this structure:

game-folder/
  title.png
  audio/
    track-name.mp3

Examples:

tekken-5/title.png
tekken-5/audio/moonlit-wilderness.mp3
street-fighter-2/title.png
street-fighter-2/audio/guile-theme.mp3
super-street-fighter-4/title.webp
super-street-fighter-4/audio/character-select.mp3
street-fighter-5-arcade-edition/title.svg
street-fighter-5-arcade-edition/audio/main-menu.mp3
sfxtekken/title.png
sfxtekken/audio/main-menu.mp3

Keep filenames lowercase and use hyphens instead of spaces.
